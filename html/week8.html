In larger react projects with many components, the repetitive typing can get pretty annoying. I found a visual studio code plugin that improves my React quality of life by adding keyboard shortcuts to automaticially create component templates. An example would be creating a new file called Example.js and typing the following:

rafce

Once the enter key is pressed, the contents of the file are replaced with the following:

import React from 'react'

const Example = () => {
  return (
   <div>

   </div>
 )
}

export default Example

I really like this extension and have used it for many of my own personal project. I like it because it just takes a lot of the setup out of the programmer's hands and allows them to focus on the actual content of the app they're building.

rafce is not the only shortcut included in this extension, there are dozens, although rafce is the one I use by far the most often. Instructions for how to download and install the plugin, as well as a list of its shortcuts and how to use it are all at the following link:

https://marketplace.visualstudio.com/items?itemName=dsznajder.es7-react-js-snippets
