CSS Animations Continued

Week 3, 9/16/2021

This week I decided to continue with CSS animations. We learned flexboxes this week, but I felt as though I could not do anything more interesting with them than what we did in class so I continued with my efforts from last week.

While the gradient background was cool last week, I felt as if it had readability issues and may be espicially bad for those with accesibiliy concerns which is why I am removing it from subsequent blog posts.

I like the idea of having some sort of animation for when the page loads, while the one I added to this page is really simple I feel as if it demonstrates the concept.

animation: fadeInAnimation ease 2s;
animation-iteration-count: 1;
animation-fill-mode: forwards;

I added the above lines to the body tag of my css file. I thought I was going to have to play with the '::before' attribute but it was really just a simple animation played once. The animation looks like this:

@keyframes fadeInAnimation {
 0% {
  opacity: 0;
 }
 100% {
   opacity: 1;
 }
}
