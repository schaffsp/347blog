React Styled Components

Week 9, 12/02/2021

This week I decided to look into styled-components for react. It is a library for react that integrates CSS into .JS files. It has some behind the scenes optimizations from what I read, but I like it because of the simplicity it provides in not having to link CSS files all the time. It is also a fairly popular library.

To install it, type the following into a terminal opened to your project directory:

npm install styled-components

Then, in any .js file you use styled components in, simply import it like so:

import styled from 'styled-components';

Here is an example in code:

const customH1 = styled.h1`
 color: blue;
`

The custome styled components are declared like any other variable, and are flexible in that any valid CSS code writen between the ` symbols will be applied to the object. The h1, in "styled.h1", is simply an HTML tag. It can be replaced with div, p, h2, and etc.

Now, somewhere lower in the file, you can type the following to use your custom styled-component.

<customH1>*SOME TEXT*</customH1>

My favorite feature of styled components is that it allows for JS to be inserted into CSS, and not just the other way around. Here is an example:

const customH1 = styled.h1`
 color: ${props.color};
`
